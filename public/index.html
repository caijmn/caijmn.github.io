炖鱼头

发表于 2022-09-04 分类于 每日一餐

今天做一道硬菜，炖鱼头。

因为没有卖胖头鱼头的，所以就买了花莲鱼头，买的时候已经从头顶部切成了两半。虽然鱼已经杀好了，不过还是要用开水烫一下外表，去除没有刮干净的鱼鳞，还要将内部的黑膜也去除一下。

清理好的鱼头就可以入锅煎了。

煎差不多之后将鱼头翻过来，加入葱段、姜段、料酒、白醋炒出香味，然后加入热水开始炖鱼。

汤烧开之后将葱段、姜段捞出，加入盐、酱油调味，然后继续炖。
等鱼头炖熟之后加入葱、姜丝、胡椒粉、白醋调出鱼汤的味道，最后再加入一些香油就做好了。

由于碗不够大，所以将鱼头单独盛出放在盘内，鱼汤用碗盛出作为配汤。

结瓜有望

2周之前，这两棵西瓜就已经开花了。

我把所有爬藤杆都用了，似乎还不够它们爬的。

在最新长出的瓜藤的末端，发现2株雌花！

由于是晚上，花已经闭合了，不过还是能看到雌花下面的小瓜，授粉之后小瓜就会慢慢长大变成大西瓜。

周日游玩

发表于 2022-08-28 分类于 吃喝玩乐

难得高温天气离去，又恰逢星期日，正好适合出门逛逛。中午吃过饭后准备来梅赛德斯奔驰文化中心里看个电影。

梅赛德斯奔驰文化中心是2010年世博会留下来的建筑，总体结构为圆形，坐落于黄浦江边，疫情之前时常举办音乐会、展览会什么的。

值得一提的是如果开的是奔驰车的话，可以享受一天仅需10元的优惠停车价格。在这种优惠力度下，来这里停车的奔驰车非常多。

电影院位于文化中心的6层，在这一层还有一条观景步道位于楼层的最外圈，步道环楼一周，首尾相接，能够观赏到360度的风景。

来到东北侧可以看到黄浦江、陆家嘴、南浦大桥汇于一景。

来到西北侧能够看到卢浦大桥。

黄焖鸡米饭

发表于 2022-08-27 更新于 2022-09-04 分类于 每日一餐

今天我们要做的是制作简便、深受大众喜爱的黄焖鸡米饭的家庭版。

首先将鸡肉切成小块，如果带骨的话，就一起剁碎。用筷子扶着的话能够避免剁手。

接下来腌制鸡肉，生抽、耗油、鸡精、盐、糖浆，基本上就是什么鲜放什么，这些腌料基本上就是黄焖鸡的全部味道了。

将鸡肉与香菇或者其他菌类在高压锅中加压3分钟左右，黄焖鸡的鸡肉和鸡汤就做好。将鸡肉和鸡汤捞出来与配菜一起再炖一小会儿，黄焖鸡的成品就做好了。用黄焖鸡米饭的汤汁拌饭，非常好吃。

在制作黄焖鸡的时候，可以一次多做一些鸡肉，接下来的几天只要每次捞出一些鸡肉再加上一些配菜炖煮一下就好了，不仅方便还可以有效清理冰箱里的剩余蔬菜。

菠萝饭

发表于 2022-08-09 更新于 2022-09-04 分类于 每日一餐

今天晚饭的主角是菠萝饭。

首先将西红柿、青椒、洋葱、胡萝卜切成小粒，和做好的米饭一起分层铺入烤盘当中，烤盘需要提前润一层油。

然后铺上火腿碎和菠萝块，并在上面再撒上一层芝士碎。

放入烤箱烘烤40~60分钟，菠萝饭就做好了。用烤箱的话比较慢，如果用炒锅翻炒的话会快很多。

最上面一层菠萝的甜味被完全释放出来，混合着芝士与火腿的香味；中间一层蔬菜的清爽夹杂着西红柿的汤汁；最下面一层米饭有炒饭的感觉。

最后再配上丝瓜鸡头米汤。

我的幼苗总是在徒长

发表于 2022-07-27 分类于 农人日记

自冬末至盛夏，阳台菜园中种植的蔬菜并没有像预想的一样呈现出硕果累累的迹象，而是向颗粒无收的方向发展，种子发芽长成幼苗后便出现了生长停滞、逐渐枯萎的现象。

造成阳台菜园产量低的最主要原因是种植过密，导致各个植株之间互相竞争生长空间，没有办法进行正常的生长。

第二个原因就是幼苗徒长，也就是幼苗徒劳无功的生长，并没有将营养用于正常发育，而是用于让茎长得更长。

当种植过密或是2棵幼苗挨得过近时，幼苗就会为了获得更好的光照条件优先生长茎，并暂缓真叶的生长。更为普遍的情况是由于阳台光照不足，幼苗为了获得更充足的光照而徒长。

上图是放在室内的西瓜幼苗，左侧的幼苗，由于光照不足产生徒长；右侧同时发芽2棵幼苗，若不即使清除掉其中1棵，2棵幼苗都会徒长。

幼苗徒长之后由于发育不协调，茎部分过长，都会出现倒伏的现象，整个苗趴在地上。过长的茎还导致植株生长困难，真叶发育迟缓，叶子容易枯黄。

上面的樱桃萝卜幼苗由于徒长后茎过长，无法支撑整个幼苗的重量，全部倒伏在土上。

由于优先生长了茎，导致真叶生长迟滞，原有的叶子也出现了枯黄。

还有的幼苗茎部出现了萎缩，萎缩后的茎就更加难以从根部供给叶子生长所需的养料。这些幼苗很可能是根部遭受了损伤，由于茎部较长，能够明显看到茎部的变化。

由于光照不足所造成的徒长是阳台环境比较难以克服的。由于房屋的存在至少挡住了一个方向的阳光，阳台环境本身就不能提供360度的光照，先天就存在劣势。当出现连续阴、雨天气时，就很难避免徒长。

放在室内的幼苗由于接受光照的角度变得更小，几乎都会产生徒长现象，当出现连续强风天气时，由于无法将幼苗放于室外，徒长现象就难以避免。

当幼苗发生徒长现象后，可以采取增加光照的措施让幼苗尽早恢复正常生长，争取仍然能够开花结果。另外即使补种一些新苗替代徒长的旧苗也是不错的办法。

上面的西瓜幼苗在徒长之后增加了光照，恢复了叶子的生长。
